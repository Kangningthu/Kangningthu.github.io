Gradient analysis of the natural image

Denoising CNNs adapt to signal structure and noise level. The first column shows part of a natural image corrupted by additive Gaussian noise. The second and third columns show the denoised output produced by a CNN. The fourth column shows an equivalent filter, equal to the gradient of an output pixel with respect to the network input, where the pixel is indicated by a red dot in the three first columns. The equivalent filters reveal that the network implements an adaptive average over regions where the intensity of the underlying clean image is approximately constant. The average adapts to the noise statistics of the input image, encompassing a larger region when the noise level increases (compare the second row with the first, or the fourth with the third), but without blurring object boundaries.

Training Code: Supervised vs Unsupervised Training

We provide the training code to compare the effect of training set size on unsupervised denoising. This figure compares the test performance of CNNs trained following the Noise2Noise framework. The CNNs are trained using two different-sized datasets of 1D piecewise-constant signals. Performance is noticeably worse for the small training set with 10 signals, but essentially the same as the supervised model for the larger training set with 500 signals.

Gradient Analysis Code

We provide inference code for the piecewise constant denoising model to show the gradient analysis. Denoising CNNs learn to perform adaptive averaging. The top image shows a 1D piecewise-constant signal corrupted by additive Gaussian noise. The image below shows the result of applying a denoising CNN to the noisy data. The three bottom images show equivalent filters, equal to the gradient of the corresponding colored entry of the denoised estimate with respect to the noisy input. The equivalent filters reveal that the CNN learns to perform approximate adaptive averages within the piecewise-constant segments of the signal, without blurring the segment boundaries.

Gradient analysis of the microscopy image

Denoising CNNs jointly exploit local and periodic structure. The first column shows a frame of real-world transmission-electron-microscopy noisy data. The second and third columns show the denoised output produced by a CNN. The fourth column shows an equivalent filter, equal to the gradient of an output pixel with respect to the network input, where the pixel is indicated by a red dot in the first three columns. The equivalent filters reveal that the CNN is able to exploit local structure via averaging, while taking into account the underlying periodic structure.
